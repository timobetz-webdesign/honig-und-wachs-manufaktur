Install and configuration
1. go to github and "import repository" (see icon "+" in the header on the right
2. copy/past kirby plainkit repo url and name the new repo
3. open "GitKraken" and clone repo from GitHub account
4. open terminal and go to project directory (e.g. /Applications/MAMP/htdocs/project-name/)
5. run: npm install bulma-start
6. start MAMP and start server
